# Prepare

```
$ export DRIBBBLE_URL='https://api.dribbble.com/v1/shots?sort=recent'
$ export DRIBBBLE_TOKEN='xxxxxxxxxx'
```

# Run

```
$ bundle install
$ rails s
```
